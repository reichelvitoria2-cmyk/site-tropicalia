# 🌴 Site Acessível sobre Tropicália 🎸 ## 📜 Descrição do Projeto Este projeto é um site simples e acessível que apresenta o movimento cultural **Tropicália**, destacando sua importância na música, arte e política brasileira dos anos 60. Pensado para ser fácil de navegar e acessível pra todo mundo, inclusive pessoas com deficiências. ## ♿ Recursos de Acessibilidade - 🔍 Menu para aumentar/diminuir fonte - 🎨 Modo alto contraste ativável - 🏷️ Textos alternativos (`alt`) em todas as imagens - 🧭 Navegação fácil com atributos ARIA - 📚 Hierarquia clara de títulos (`

`, `

`, etc.) - 🎨 Contraste de cores adequado para facilitar a leitura ## 💻 Tecnologias Usadas - HTML5 - CSS3 - JavaScript (pra funcionalidades acessíveis) --- ## 🚀 Como usar esse repo 1. Clone ou baixe o projeto 2. Abra o `index.html` no navegador 3. Use o menu de acessibilidade pra ajustar do seu jeito --- Made with ❤️ e muito groove da Tropicália!
